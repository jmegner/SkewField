SkewField.py contains the classes and functionality for handling letters,
words, polynomials, relations, and so on.

JPolyMat.py contains the code to diagonalize matrices of SkewFieldPolynomials
and relations to compute delta1.

knotX_Y.txt type files contain the SkewFieldPolynomials and relations for
various knots.  They are inputs to JPolyMat.py.

You can ignore KittySupplements.py and ExampleSkewFieldUser.py; their
usefulness has passed.
